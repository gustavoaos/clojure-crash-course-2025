### Clojure tips

Clojure uses Normal Polish Notation
```clojure
(operator/function arguments)
```

Converting this expression ``` 7 + 5 - (8 * 3 * 5)``` to Clojure Normal Polish Notation will result in
```clojure
(- (+ 7 5) (* 8 3 5))
```
